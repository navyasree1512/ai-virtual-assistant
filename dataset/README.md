# Google Speech Commands Dataset

## Dataset Name
Google Speech Commands Dataset v0.02

## Source
https://www.kaggle.com/datasets/yashdogra/speech-commands

## Purpose
Used for speech recognition and voice command detection in the AI Virtual Assistant.

## Total Size
Approximately 3.34 GB

## Data Type
Audio (.wav)

## Features

- Audio File
- Command Label
- Speaker ID
- Background Noise

## Commands Included

- Yes
- No
- Up
- Down
- Left
- Right
- Go
- Stop
- On
- Off
- Zero–Nine

## Usage

The dataset is used for recognizing spoken commands that are processed by the virtual assistant.
